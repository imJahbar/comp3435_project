About This Project

RSSH (Road and Smart Solutions for Holes) is a web-based pothole monitoring system developed for COMP3435 User Interface Design. It allows regular users a personalised view of road conditions on roads they care about. It also provides maintenance crews to log pothole observations, estimate repair costs, and generate reports. The system is designed to be accessible to a number of groups with disabilities but it caters specifically to elderly and persons with dyslexia with a number of built in visual customizations available on the accessibility tab.
